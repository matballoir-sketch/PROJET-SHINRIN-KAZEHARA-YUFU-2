MISE EN LIGNE SUR GITHUB PAGES
1. Créez un nouveau dépôt GitHub.
2. Importez tous les fichiers en conservant exactement les dossiers.
3. Ouvrez Settings > Pages.
4. Source : Deploy from a branch.
5. Branche : main, dossier : /root.
6. Enregistrez. Le site sera disponible après quelques minutes.

IMPORTANT
La musique YouTube démarre après le clic sur « Entrer dans son histoire », conformément aux règles des navigateurs.
